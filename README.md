# 五子棋游戏

一个基于 HTML5 Canvas 实现的简洁五子棋游戏，支持双人对战。

## 功能特点

- 🎮 双人对战模式
- 🎨 精美的渐变背景和棋盘设计
- ✨ 流畅的游戏体验
- 🔄 一键重新开始
- 📱 响应式设计

## 游戏规则

- 15×15 标准棋盘
- 黑方先手，白方后手
- 横、竖、斜任意方向连成五子即获胜
- 点击棋盘交叉点落子

## 快速开始

直接在浏览器中打开 `index.html` 文件即可开始游戏。

```bash
# 克隆仓库
git clone https://github.com/guoxiaozhen0304/wuziqi.git

# 进入目录
cd wuziqi

# 在浏览器中打开
start index.html  # Windows
open index.html   # macOS
```
